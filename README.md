# Shramana Project Site

This repository publishes the static public website for Shramana Project.

## Publish

1. Export the site from the private app repo:

   ```bash
   ruby Scripts/export_hosted_site.rb
   ```

2. Copy or push the exported `/tmp/shramana-site` bundle to this public repo.

3. Enable GitHub Pages using the `pages` workflow.

## URLs

- Marketing URL: `https://ninjatomonline.github.io/shramana-site/`
- Support URL: `https://ninjatomonline.github.io/shramana-site/support.html`
- Privacy Policy URL: `https://ninjatomonline.github.io/shramana-site/privacy.html`

The App Store URL should be added only after the public listing exists.
