Press

Shramana Project is an iPad-only puzzle pilgrimage about guiding many monks so one can pass through the gate.

Short Pitch

A Buddhist-themed side-view puzzle game inspired by classic crowd-routing design, built with SpriteKit, SwiftUI, Codable level data, and generated PixelLab art.

Facts

  • Platform: iPadOS 17 and later
  • Orientation: landscape left and right
  • Campaign: 20 levels, four chapters
  • Core loop: assign limited monk abilities, solve local routing puzzles, save exactly one monk when the level demands it
  • Status: local app project and hosted site package in repo, public App Store page pending
