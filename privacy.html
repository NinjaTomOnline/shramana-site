Privacy

Plain-language policy for the current local app architecture.

Data collected

The current Shramana Project build does not include account creation, analytics, advertising SDKs, networking, cloud sync, or third-party telemetry systems.

Data stored on device

The app stores local gameplay progress, level completion, medals, best-time data, settings, and recovery information for crash-safe saves.

Data shared

No personal gameplay data is shared by the current app code. If a future release adds optional support logs or cloud features, this policy should be updated before those features ship.

Children and families

The game is designed as a puzzle experience. The repo does not include chat, public profiles, user-generated content, or social features.

Contact

Use the support page for current contact guidance until the public App Store listing and official support address are configured.
